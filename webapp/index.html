API calls available:

  • Get list of all accounts
    /api/listAccounts
  • Get account by id : /api/account?id=[account id]
    /api/account?id=1
  • Create account : /api/createAccount?name=[account name]&balance=[initial balance]
    /api/createAccount?name=Max&balance=1000
  • Transfer money from one account to another : /api/transfer?fromId=[orderer id]&toId=[counter-party id]&amount=[amount to be transfered]
    /api/transfer?fromId=4&toId=1&amount=300
